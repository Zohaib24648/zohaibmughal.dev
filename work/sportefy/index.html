Sports Venue Booking & Matchmaking

A B2B2C sports platform live in Pakistan — consumer Flutter app, Next.js admin dashboard and a versioned NestJS API. Book venues, find matches, pay.

RoleFull-Stack + Mobile Engineer
TimelineMar 2025 — Dec 2025 · retainer
StatusLive — App Store & Play Store
ShapeB2B2C · app + admin + API
The Brief

Book venues, find matches, pay.

Players book courts, join matches and pay from a consumer app; venue operators and platform staff run everything from an admin dashboard — all on one versioned API that never breaks the mixed-version mobile install base.

Consumer Flutter app — booking, slots, QR check-in & wallet, offline-first with request replay
Realtime matchmaking & match chat — Supabase Realtime + Postgres row-level security
Next.js admin dashboard over a versioned /v1 API, one shared RBAC matrix (5 platform roles + venue-scoped staff)
Payments domain — pluggable gateway adapter (Paymob), wallet / credits ledger, promo codes, PDF booking confirmations
