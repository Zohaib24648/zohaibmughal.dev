The Brief

Staff certified crew, without the chaos.

Event companies need to staff certified freelance crew for live events — with conflict-free scheduling, compliant pay and automated payouts — on web and mobile alike. Hyss handles the whole loop: match, schedule, work, invoice, pay.

Wizard-driven event creation & recurring shifts
Competency-based matching with conflict detection
Freelancer calendars, time-off requests & bidirectional Google / Apple calendar sync
Stripe Connect financial core — payout onboarding, hybrid seat + usage-metered billing, per-invoice checkout links
